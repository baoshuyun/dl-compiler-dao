# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by emailing the project maintainers
or opening a private vulnerability report on GitHub.

Do **not** open a public issue for security vulnerabilities.

## Scope

This policy covers all three sub-projects:
- `mini-dl-compiler` — compiler passes and code generation
- `NPU_Soft_Hard_Stack` — frontend and backend scheduling
- `NPU_Project` — Verilog NPU hardware design

## Response Timeline

- Acknowledgment: within 48 hours
- Fix: within 7 days of confirmation
- Disclosure: after fix is released
